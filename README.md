# Plan My Yatra

A simple travel trip planning website that helps users discover destinations and plan their trips.

## Features

- **Home Page**: Welcome page with navigation
- **User Authentication**: Sign up and login functionality
- **Destinations**: Browse and explore travel destinations
- **Trip Planning**: Plan your trip with customized options
- **About**: Learn more about the platform
- **Contact**: Get in touch with the team

## Technologies Used

- HTML5
- CSS3
- Responsive Design

## Getting Started

1. Clone or download the project
2. Open `index.html` in your web browser
3. Navigate through the site using the navigation menu

## Pages

- **Home**: Main landing page with overview
- **Destinations**: Explore popular travel destinations
- **Plan Trip**: Create and customize your travel itinerary
- **About**: Information about the platform
- **Contact**: Contact form to reach out
- **Login/Signup**: User authentication pages
